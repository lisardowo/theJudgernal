## What's inside

### Home
The landing page. A little penguin greets you and a dialogue box introduces the site. After clicking the arrow you get navigation options to all sections.

### What I bring to the table (`/AboutMe`)
A terminal-style PowerShell window that describes who I am and what skills / things I bring to the table.

### I'm a nerd (`/Projects`)
A grid of project cards, each with a title, description, and a link to its repository.

### Blogs (`/Blogs`)
A blog reader with two tabs — written posts and video recommendations. Each post has a thumbnail, timestamp, content, and a closing image.

### 88x31 (`/88x31`)
A compilation of 88×31 pixel web buttons I found while exploring the internet. Hovering/clicking the penguin in the corner opens the nav menu. The buttons say something about me (at least I think so).

## Tech stack

- **React 19** + **TypeScript**
- **Vite** (bundler)
- **React Router** (HashRouter for GitHub Pages compatibility)
- **GSAP** (animations)
- **gh-pages** (deployment)

## Run locally

```bash
npm install
npm run dev
```

## Deploy
